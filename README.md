# mm2image

Saves [mermaid diagrams](https://github.com/mermaid-js/mermaid) as an image

## Installation

    $ python3 -m venv venv
    $ source venv/bin/activate
    (venv) $ pip install -r requirements.txt
    (venv) $ pip install -r dev-requirements.txt
    (venv) $ inv -l  # show available tasks
